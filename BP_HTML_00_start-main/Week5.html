HTML & CSS - Typografie, Kleur en Ruimte

Coderen

In lesweek 5 leerden we op basis van HTML en CSS een website maken met Visual Studio Code. In de website zetten we onze projectdocumentatie en het doel is om inzicht te krijgen in de structuur van webpagina's, hoe inhoud wordt opgebouwd met markup, en hoe eenvoudige basisopmaak kan worden toegepast.

In een HTML plaatsen we elementen zoals koppen, paragrafen, lijsten en links, en in CSS wordt kleur, lettertypes en indeling toegevoegd. We begonnen met het maken van een account op de websites GitHub en Netlify, daarna hebben we leren coderen. Visual Studio Code zit tussen mappen en bestanden, het resultaat van webpagina kan je zien in de browser. Eerst raakte ik totaal in de war over hoe dit werkt, maar uiteindelijke lukte het wel, momenteel ben ik zeer tevreden over het resultaat.
